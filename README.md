# Diabetes Prediction using KNN

This project predicts whether a patient has diabetes using machine learning.

Algorithm used:
K-Nearest Neighbors (KNN)

Tools:
Python
Pandas
NumPy
Scikit-learn

Steps performed:
1. Data loading
2. Data preprocessing
3. Train-test split
4. Feature scaling
5. Model training
6. Prediction

Dataset:
Pima Indians Diabetes Dataset
